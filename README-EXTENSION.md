# Video Speed Controller - Chrome Extension

A Chrome extension that adds a sleek, draggable speed controller to any video on the web.

## Features

- **Speed Range**: Control playback from 0.5x to 16x
- **Draggable Interface**: Move the controller anywhere on the screen
- **Preset Buttons**: Quick access to common speeds (0.5x, 1x, 1.5x, 2x, 3x)
- **Keyboard Shortcut**: Press `Ctrl+Shift+S` to show/hide the controller
- **Auto-Detection**: Automatically appears when videos are detected on any webpage
- **Modern Design**: Beautiful gradient UI with smooth animations

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" in the top right corner
3. Click "Load unpacked"
4. Select the folder containing these extension files
5. The extension is now installed!

## Usage

1. Navigate to any webpage with a video (YouTube, Vimeo, Netflix, etc.)
2. The speed controller will automatically appear in the top-right corner
3. Use the slider to adjust speed from 0.5x to 16x
4. Click preset buttons for quick speed changes
5. Drag the controller by clicking and holding the header
